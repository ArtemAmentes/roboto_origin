## О проекте

**РОБОТ** — первый полностью российский открытый проект по созданию робота-андроида с собственным программным обеспечением и отечественной компонентной базой.

### Что мы создаём

| Направление | Описание |
|-------------|----------|
| **Российское ПО** | Собственный SDK, драйверы, RL-алгоритмы обучения, AI-система управления |
| **Отечественные компоненты** | Моторы, датчики, электроника от российских производителей |
| **Открытая документация** | Полная конструкторская документация на русском языке |
| **AI-разработка** | Команда AI-агентов (Claude + Gemini) для автономной разработки ПО |
| **Образование** | Техникум робототехники, курсы, методические материалы |

### Ключевые особенности

- **100% открытый исходный код** — GPL v3, свободное использование и модификация
- **Полный производственный цикл в России** — от проектирования до серийного выпуска
- **Camera-in-the-Loop** — AI-система автономно улучшает ПО, наблюдая за роботом через камеру
- **Импортонезависимость** — постепенный переход на российские комплектующие

### Благодарности

Проект стартовал на основе открытых материалов китайского сообщества [RoboParty](https://github.com/Roboparty/roboto_origin). Мы благодарим китайских коллег за открытость и вклад в мировое развитие робототехники. Текущая версия полностью переработана и адаптирована для российских условий с собственной архитектурой и компонентной базой.
