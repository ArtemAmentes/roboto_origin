# Лазерная гравировка

> Источник: [https://watako-lab.com/2018/09/05/lasermarking1st/](https://watako-lab.com/2018/09/05/lasermarking1st/)
> Источник: [https://watako-lab.com/2017/12/11/3dprintlaser/](https://watako-lab.com/2017/12/11/3dprintlaser/)

## Используемый лазерный гравёр

| Параметр | Значение |
|----------|----------|
| **Модель** | Meterk DK-BL |
| **Мощность** | 1500 мВт |
| **Сайт производителя** | [https://www.meterk.com/](https://www.meterk.com/) |
| **Цена** | ~12 700 иен на Amazon (со скидкой CyberMonday ~6000 иен дешевле) |
| **Размеры** | ~160 x 145 x 200 мм |

## Характеристики гравировки

| Параметр | Значение |
|----------|----------|
| **Максимальный размер изображения** | 550 x 550 px |
| **Область гравировки** | 42 x 42 мм |
| **Разрешение** | ~350 dpi |
| **Время гравировки** (пример) | ~10 минут |
| **Входной формат** | JPEG-изображение |

## Материал

Гравировка выполнялась на **MDF-панелях** (древесно-волокнистая плита средней плотности). Размер заготовки: ~50 x 50 мм.

Побочные эффекты: небольшой дым и запах горелого дерева.

## Доработка платформы

Для удобной фиксации заготовки создано приспособление из:
- Универсальных крепёжных элементов **Tamiya** (Universal Metal Fittings + Long Universal Arm)
- **Неодимовых магнитов** из 100-иенного магазина (Daiso)

## Применение

- Декоративные таблички для роботов (например, подставка для Дроид-куна на выставках)
- Маркировка деталей и панелей

## Инструмент для раскроя плат

| Параметр | Значение |
|----------|----------|
| **Модель** | Proxxon Circular Saw Table No.27006 |
| **Цена** | ~10 000+ иен на Amazon |
| **Особенности** | Компактный, тихий, возможность подключения пылесоса |
| **Диск** | Алмазный диск (приобретается отдельно) |

Используется для раскроя печатных плат и MDF-панелей.
