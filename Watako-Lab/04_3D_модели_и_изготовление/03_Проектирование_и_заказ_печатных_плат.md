# Проектирование и заказ печатных плат (PCB)

## Обзор процесса

3 шага: **проектирование** → **конвертация в Gerber** → **заказ у производителя**.

## 1. Проектирование

### Софт: EAGLE

| Параметр | Значение |
|----------|----------|
| **Бесплатная версия** | До 80 см² площади платы |
| **Автотрассировка** | Да (автоматическая разводка дорожек) |
| **Учебный ресурс** | [AKIRACING.com](http://akiracing.com/2017/05/27/eagle_tutorial/) — основы за ~1 час |

### Параметры разводки

| Параметр | Стандарт PCBWay |
|----------|-----------------|
| Мин. диаметр отверстия | 0.3 мм |
| Мин. зазор между дорожками | 6 mil (0.1524 мм) |
| Ширина дорожки питания | Зависит от тока (для 2A — расширенная) |

Ресурс по ширине дорожек: [Noise-Counterplan.com](http://www.noise-counterplan.com/article/14951064.html)

## 2. Конвертация в Gerber

Формат **Gerber** — отраслевой стандарт для производства PCB.

### Состав файлов Gerber

| Расширение | Содержимое |
|------------|-----------|
| `.GBL` | Медные дорожки (нижний слой) |
| `.GBO` | Шелкография (нижний слой) |
| `.GBS` | Паяльная маска (нижний слой) |
| `.GTL` | Медные дорожки (верхний слой) |
| `.GTO` | Шелкография (верхний слой) |
| `.GTS` | Паяльная маска (верхний слой) |
| `.TXT` | Данные сверления |
| `.GML` | Контур фрезеровки |

Файлы упаковываются в ZIP и загружаются на сайт производителя.

### Просмотр Gerber

Софт: [Gerbv (Gerber Viewer)](https://sourceforge.net/projects/gerbv/) — бесплатный.

> **Нюанс:** Данные сверловки могут отображаться со смещением — нужно вручную указать формат координат.

### Проверка на бумаге

Распечатать Gerber 1:1 на принтере → приложить реальные компоненты → проверить, что всё помещается.

## 3. Заказ у производителя

### Производитель: PCBWay

| Параметр | Значение |
|----------|----------|
| **Сайт** | [https://www.pcbway.com/](https://www.pcbway.com/) |
| **Стоимость** | 10 плат = **$5** (~50 центов/шт!) |
| **Доставка** | E-packet (самая дешёвая) |
| **Итого** | **~$12** за 10 плат с доставкой |
| **Срок** | ~2 недели (с дешёвой доставкой) |
| **Проверка** | Автоматическая проверка DRC (~1 час) |

### Настройки заказа

- Цвет паяльной маски и шелкографии выбирается (белая маска + чёрный шёлк — стильно, но видны следы флюса)
- Финишное покрытие: HASL (по умолчанию), может быть заменено на ENIG без доплаты

### Трюк: панелизация (面付け)

Если несколько мелких плат, их можно **объединить на одном листе** (до ~10 x 10 см) и заказать как одну плату. Стоимость не меняется!

Пример: 3 разных типа плат × 4 комплекта = 12 плат по цене 1 заказа.

**Раскрой** выполняется мини-циркуляркой Proxxon No.27006 с алмазным диском.

## Результаты заказа

- Качество: отличное, без дефектов, сложные контуры выполнены точно
- Проводимость: все дорожки исправны
- Вывод автора: «Раньше бы узнал — раньше бы заказал»

### Очистка от флюса

Средство для удаления флюса — наносится на коричневые пятна, плата становится чистой.
