# Дроид-кун Ver.2 — говорящий IoT-робот

> Источник: [https://watako-lab.com/2017/12/01/droid_ver2_introduction/](https://watako-lab.com/2017/12/01/droid_ver2_introduction/)
> Английская версия: [https://watako-lab.com/2017/12/01/about_droid-kun_ver2/](https://watako-lab.com/2017/12/01/about_droid-kun_ver2/)

## Обзор проекта

**Дроид-кун Ver.2** (ドロイド君) — робот-маскот Android от Google, оснащённый множеством датчиков и подключённый к умному дому.

### Концепция

Робот, который не просто «полезен», а создаёт ощущение тёплого общения. Интеграция с Google Home для управления бытовой техникой, общение через сенсоры, танцы.

*Примечание: «Дроид-кун» — неофициальное название маскота Android от Google.*

## Ключевые функции

### 1. Интеграция с Google Home

- Управление бытовой техникой через голосовые команды
- Технологии: IFTTT + MQTT
- Робот является посредником между Google Home и устройствами в доме

### 2. Дневник роста

| Этап | Функция | Описание |
|------|---------|----------|
| 1 | Чтение чисел + датчик температуры/влажности | Робот озвучивает числа и показания датчиков |
| 2 | Дальномер + микрофон | Обнаружение объектов и запись звука |
| 3 | Распознавание речи v1 | Первая версия + бонусный танец |
| 4 | Распознавание речи v2 | Улучшенная версия + управление LED-подсветкой |
| 5 | Пульт ДУ | Управление с ИК-пульта |
| 6 | Отслеживание пальца | Интерактивная игра с пальцем (2 версии) |
| 7 | Bluetooth-динамик | Воспроизведение музыки через Bluetooth |

### 3. Совместные действия

- Два робота Дроид-кун танцуют синхронно

### 4. Дроид-кун Ver.2 Light

- Облегчённая версия без подставки

## Планы развития

| Функция | Описание |
|---------|----------|
| Расширение интерактива | Больше вариантов реакций с использованием сенсоров |
| Чат-управление | Управление роботом через Line-подобный мессенджер со смартфона |
| Энергосбережение | Переход в спящий режим по данным сенсоров |
| Учебная платформа | Создание простого окружения программирования (уровень начальной школы) |
| Общение между роботами | Игра в «ширитори» (словесная цепочка), синхронный танец |
| Улучшение распознавания речи | Рассматриваются Julius (свободная система) или Cloud Speech API |
