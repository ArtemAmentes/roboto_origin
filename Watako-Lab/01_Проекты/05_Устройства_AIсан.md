# Аппаратная конфигурация робота AIсан

> Источник: [https://watako-lab.com/2020/02/11/aisan_devices/](https://watako-lab.com/2020/02/11/aisan_devices/)

## Обзор

Описание модулей, добавленных и изменённых по сравнению с базовым PremaidAI для создания робота AIсан. Модифицированные зоны: **голова**, **грудная часть**, **стопы**.

## Схема соединения модулей

Модули, выделенные оранжевой рамкой — добавленные/изменённые компоненты. Наибольшие изменения — в голове (робот буквально «поумнел»).

## Голова: микроконтроллер, камера, ToF-сенсоры, динамик

Подключение сложное: множество проводов, но I2C и UART стабильно работают.

### Микроконтроллер + камера для распознавания изображений

| Компонент | Описание |
|-----------|----------|
| **Sipeed MAixBiT** | Установлен на место бывшего Bluetooth-модуля |
| **KPU** | Встроенный блок нейросетевых вычислений для YOLO-детектирования в реальном времени |
| **Камера** | Широкоугольный объектив (fish-eye) на длинном шлейфе, встроен в отверстие на лбу |

Ранее: [исследование MAixBit](https://watako-lab.com/2019/08/11/maix_robot_base/)

### ToF-сенсоры в «ушах»

- По одному ToF-сенсору в каждом «ухе»
- Установлены под разными углами:
  - Один смотрит вниз (ноги)
  - Другой смотрит вперёд (препятствия)
- «Уши» напечатаны на 3D-принтере
- Голова покрашена чёрным лаком

### Динамики (x2) внутри лица

| Компонент | Описание |
|-----------|----------|
| **Усилитель** | I2S-подключение |
| **Динамики** | 2 шт., встроены в лицевую панель |
| **Микрофон** | Планируется добавить для распознавания речи |

## Грудная часть: модуль приёмника пульта ДУ

- Приёмник для беспроводного контроллера **VS-C3** (Vstone)
- Подключён через **SPI** к штатной плате управления PremaidAI
- Обеспечивает ручное управление ходьбой и действиями

## Стопы: уменьшение размера

- Штатные стопы слишком большие для ходьбы
- **Шаг 1:** Стопы перевёрнуты (перед/зад) и задняя часть обрезана
- **Шаг 2:** Изготовлены минимальные стопы из MDF для соответствия регламенту ROBO-ONE auto

## Ключевые решения

Главная сложность проекта — **уместить все компоненты, не нарушая внешний вид PremaidAI**. Пришлось скрытно обрезать детали, сверлить отверстия и проявлять изобретательность с упаковкой.
