## 9-осевой датчик MPU9250

> Источники: серия статей по 9-осевому датчику

| Параметр | Значение |
|----------|----------|
| **Тип** | 9-осевой IMU (инерциальный модуль) |
| **Состав** | 3-осевой акселерометр + 3-осевой гироскоп + 3-осевой магнитометр |
| **Встроен в** | M5Stack Gray |
| **Протокол** | I2C / SPI |

### Применение

- Определение ориентации (Roll/Pitch/Yaw)
- Стабилизация ходьбы (гироскопическая коррекция)
- Компас (определение направления)
- Сенсорное слияние (complementary filter)

Подробные математические выкладки: см. папку `02_Математика_и_формулы/`.

## ToF-дальномер VL53L0X

> Источник: статьи о Фоло-кун, Хоппер-кун, AIсан

| Параметр | Значение |
|----------|----------|
| **Принцип** | Time-of-Flight (время пролёта лазера) |
| **Дальность** | До ~2 м |
| **Протокол** | I2C |

### Применение по проектам

| Проект | Количество | Назначение |
|--------|-----------|-----------|
| AIсан | 2 (в ушах) | Обнаружение препятствий (пол + фронт) |
| Хоппер-кун | 3 (в голове) | Обнаружение стен, планирование маршрута, построение карты |
| Колон | 2+ | Отслеживание объектов, SLAM, защита от падения |
| Фоло-кун | 1+ | Обнаружение объектов |
