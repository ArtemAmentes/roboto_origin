# Прочее оборудование, модули и инструменты

> Собрано из различных статей Watako-Lab

## Коммуникационные модули

### Беспроводной контроллер VS-C3

| Параметр | Значение |
|----------|----------|
| **Производитель** | Vstone |
| **Тип** | Беспроводной контроллер (приёмник + передатчик) |
| **Протокол** | SPI |
| **Применение** | Ручное управление AIсан, Хоппер-кун |
| **Страница** | [https://www.vstone.co.jp/products/vs_c3/index.html](https://www.vstone.co.jp/products/vs_c3/index.html) |

### Bluetooth-модуль

Штатный модуль PremaidAI — перемещён в затылок при модификации AIсан (освободил место для MAixBiT).

## Звуковые модули

### I2S-усилитель + динамики

| Параметр | Значение |
|----------|----------|
| **Протокол** | I2S (цифровой аудиоинтерфейс) |
| **Количество** | 2 динамика (AIсан — в лицевой панели) |
| **Применение** | Синтез речи, звуковые эффекты |

### Bluetooth-динамик

Дроид-кун v2 поддерживает воспроизведение музыки через Bluetooth A2DP.
