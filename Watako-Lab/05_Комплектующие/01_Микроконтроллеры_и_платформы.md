# Микроконтроллеры и платформы разработки

> Собрано из различных статей Watako-Lab

## ESP32 (ESP-WROOM-32)

> Источник: [https://watako-lab.com/2017/12/07/pedestal-2/](https://watako-lab.com/2017/12/07/pedestal-2/)

**Основной микроконтроллер** большинства проектов Watako-Lab.

| Параметр | Значение |
|----------|----------|
| CPU | 32-бит, 240 МГц, двухъядерный (Xtensa LX6) |
| SRAM | 512 КБ |
| Беспроводная связь | WiFi + Bluetooth v4.2 |
| Периферия | PWM x16, touch-сенсор, магнитный сенсор, низкошумящий усилитель, SDIO/SPI, UART, I2C, I2S |
| Гибкость I/O | Высокая свобода маппинга пинов |
| **Цена** | **~700 иен** (чип) / **~1480 иен** (Dev Board) |

### Применение в проектах

| Проект | Роль ESP32 |
|--------|-----------|
| Дроид-кун v2 | Основной микроконтроллер (управление сервоприводами, звук, IoT) |
| Колон | Основной микроконтроллер (навигация, SLAM, сенсоры) |
| Хоппер-кун | 2 × ESP32 (мозг + привод) + M5Atom |
| AIсан | Управление ходьбой и сервоприводами |

### Где купить

Доступен на Amazon Japan, AliExpress, Akizuki Denshi и др. Рекомендованы модули с разведённой обвязкой (dev board).

## M5Stack Gray

> Источник: [https://watako-lab.com/2018/05/07/m5stack_gray/](https://watako-lab.com/2018/05/07/m5stack_gray/)

| Параметр | Значение |
|----------|----------|
| Основа | ESP32 |
| Дисплей | Цветной LCD |
| Звук | Встроенный динамик |
| Хранение | SD-карта |
| **9-осевой датчик** | **MPU9250** (3-осевой акселерометр + гироскоп + магнитометр) |
| Размер | ~5 x 5 см |

### Применение

Основная платформа для серии «Управление 9-осевым датчиком» — все математические конспекты (RPY, акселерометр, магнитометр, гироскоп) написаны на основе M5Stack Gray.

## Sipeed MAixBiT (Kendryte K210)

> Источник: [https://watako-lab.com/2019/08/11/maix_robot_base/](https://watako-lab.com/2019/08/11/maix_robot_base/)

| Параметр | Значение |
|----------|----------|
| Процессор | RISC-V 64-бит, двухъядерный, 400 МГц |
| **KPU** | Встроенный нейросетевой ускоритель |
| Нейросети | YOLO (детектирование объектов в реальном времени) |
| Камера | Подключаемая (fish-eye для AIсан) |

### Применение

| Проект | Роль |
|--------|------|
| AIсан | Распознавание объектов (панели для ROBO-ONE auto) |
| Хоппер-кун | MAixM1n — распознавание лиц (~11 fps), CNN-детектирование |

---

## Sipeed MAixM1n

Компактный модуль на базе K210. Используется в Хоппер-кун для:
- Распознавания лиц
- CNN-детектирования объектов (обученные модели для конкретных роботов)
