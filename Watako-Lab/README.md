# Watako-Lab — Конспекты по робототехнике

> Источник: [https://watako-lab.com/](https://watako-lab.com/)
> Автор оригинала: Watako (@WatakoLab)
> Перевод и структурирование: конспект-база

## Описание

Watako-Lab — лаборатория электронных самоделок, специализирующаяся на робототехнике.
Основные направления: двуногие роботы, балансирующие роботы (обратный маятник),
сенсорные системы (9-осевые датчики), машинное зрение, SLAM и навигация.
