# Обзор проекта

Национальная платформа российской робототехники — общая концепция, направления работ и дорожная карта.

## Содержание раздела

- [Структура работ](структура_работ.md) — четыре направления (железо, софт, интеграция, кадры), work packages
- [Дорожная карта](дорожная_карта.md) — этапы 1 и 2, финансирование, критерии успеха
- [Исследование рынка гуманоидов](исследование_рынка_гуманоидов.md) — что продаётся vs только разработка, цены, инвойсы, DIY
- [Компонентная база и поставки](компонентная_база_и_поставки.md) — российские магазины, заводы, цепочки поставок до Солнечногорского района
- [Организация трёх блоков](организация_трех_блоков.md) — материальная база, софт, образование в колледжах

## Ключевые тезисы

- **Три столпа:** Железо (модульность, импортозамещение) → Софт (SDK, LLM-агент) → Образование (кадры, кафедры ВУЗов)
- **Результаты:** Открытый SDK · Открытая номенклатура · Открытое образование
- **Москва** — центр компетенций и будущее серийное производство роботов
