# Обзор проекта

**Национальная платформа российской робототехники** — госзаявка на субсидию ~10 млрд ₽ для создания полного цикла производства гуманоидных роботов в Москве.

## Ключевые тезисы

| Столп | Содержание |
|-------|------------|
| **Железо** | Модульность, импортозамещение, сборочная мастерская |
| **Софт** | Открытый SDK, LLM-агент, интеграция с MES/SCADA |
| **Образование** | Кадры, кафедры в ВУЗах, колледжи СПО |

**Результаты проекта:**
1. Открытый программный пакет SDK
2. Открытая номенклатура узлов и агрегатов
3. Открытое образование и подготовка кадров

**Москва** — центр компетенций и будущее серийное производство роботов.
