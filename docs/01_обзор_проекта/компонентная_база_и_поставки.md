# Компонентная база робототехники: российские магазины, заводы и цепочки поставок до Солнечногорского района

Документ описывает создание компонентной базы на основе данных российских интернет-магазинов и сайтов заводов. Включает ссылки на поставщиков и проработку цепочек поставок до Московской области, Солнечногорский район.

---

## 1. Структура компонентной базы

Типовой робототехнический комплекс (манипулятор, гуманоид, мобильная платформа) включает следующие категории компонентов:

| Категория | Подкатегории | Примеры номенклатуры |
|-----------|--------------|----------------------|
| **Приводы** | Сервоприводы, мотор-редукторы, шаговые двигатели | Dynamixel, MG996r, HK15298B, J60/J80/J100 |
| **Контроллеры** | Одноплатные ПК, микроконтроллеры, платы расширения | Raspberry Pi, Arduino, ESP32, STM32, Iskra |
| **Датчики** | Энкодеры, IMU, дальномеры, камеры, LiDAR | MPU6050, VL53L0X, Kinect, RealSense |
| **Питание** | Батареи, БП, преобразователи | LiPo, LiFePO4, 6V/12V/24V БП |
| **Механика** | Крепёж, редукторы, валы, подшипники | DIN 7985, шестерни, ремни GT2 |
| **Корпуса и каркасы** | 3D-печать, металлоконструкции, провода | PLA/PETG/ABS, профиль 2020, ПуГВ |
| **Производство** | Печатные платы, SMD-монтаж | Заказ PCB, контрактная сборка |

## 4. Цепочки поставок до Солнечногорского района

### 4.1. География

**Солнечногорск** — город в Московской области, ~65 км к северо-западу от Москвы по Ленинградскому шоссе. Солнечногорский муниципальный район включает г. Солнечногорск, пгт Поварово, Ржавки, Менделеево и др.

### 4.4. Рекомендации по оптимизации поставок в Солнечногорск

| Задача | Рекомендация |
|--------|--------------|
| Мелкие компоненты (<15 кг) | ChipDip + СДЭК/5Post до ПВЗ Солнечногорска |
| Тяжёлые сервоприводы, моторы | DMOTOR/ИнноДрайв + ПЭК до ул. Красная 161/19 |
| Оптовые заказы (>25 000 ₽) | ChipDip — бесплатная доставка до ПВЗ в Солнечногорск |
| Печатные платы и сборка | РАСМО (Зеленоград) — близко, бесплатная доставка |
| 3D-детали, корпуса | Локальные сервисы Москвы + СДЭК/курьер |
| Срочность | Самовывоз из Москвы (ChipDip, Амперка) — ~1,5 ч в одну сторону |

## 7. Ограничения и риски

| Риск | Меры |
|------|------|
| Дефицит импортных компонентов | Дублирование поставщиков, российские аналоги (Iskra вместо Arduino) |
| Ограничения ТК по весу/габаритам | ПЭК, Деловые Линии для грузов >25 кг |
| Аккумуляторы, опасные грузы | Уточнять у СДЭК/Почты ограничения, возможна только наземная доставка |
| Сроки из регионов (СПб, Пермь) | Заказывать заранее, 3–7 дней |
