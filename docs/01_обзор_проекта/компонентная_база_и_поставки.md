# Компонентная база робототехники: российские магазины, заводы и цепочки поставок до Солнечногорского района

Документ описывает создание компонентной базы на основе данных российских интернет-магазинов и сайтов заводов. Включает ссылки на поставщиков и проработку цепочек поставок до Московской области, Солнечногорский район.

---

## 1. Структура компонентной базы

Типовой робототехнический комплекс (манипулятор, гуманоид, мобильная платформа) включает следующие категории компонентов:

| Категория | Подкатегории | Примеры номенклатуры |
|-----------|--------------|----------------------|
| **Приводы** | Сервоприводы, мотор-редукторы, шаговые двигатели | Dynamixel, MG996r, HK15298B, J60/J80/J100 |
| **Контроллеры** | Одноплатные ПК, микроконтроллеры, платы расширения | Raspberry Pi, Arduino, ESP32, STM32, Iskra |
| **Датчики** | Энкодеры, IMU, дальномеры, камеры, LiDAR | MPU6050, VL53L0X, Kinect, RealSense |
| **Питание** | Батареи, БП, преобразователи | LiPo, LiFePO4, 6V/12V/24V БП |
| **Механика** | Крепёж, редукторы, валы, подшипники | DIN 7985, шестерни, ремни GT2 |
| **Корпуса и каркасы** | 3D-печать, металлоконструкции, провода | PLA/PETG/ABS, профиль 2020, ПуГВ |
| **Производство** | Печатные платы, SMD-монтаж | Заказ PCB, контрактная сборка |

## 7. Механические компоненты и CNC-обработка

### 7.1. Проблема

В BOM проекта много деталей, требующих:
- **CNC-обработка алюминия** — рычаги, пластины, фиксаторы
- **Подшипники специальные** — радиальные шарнирные, карданные, упорные игольчатые
- **Моторы DM-J10010L, DM4340p** — китайские, требуют посредника

Для электроники ссылки есть (ChipDip, Амперка), для механики — нет.

```
┌─────────────────────────────────────────────────────────────────────────────┐

│ 4. Посредник получает на склад, консолидирует                               │
```

## 10. Ограничения и риски

| Риск | Описание | Меры |
|------|----------|------|
| **Дефицит импортных компонентов** | Санкции, уход поставщиков | Дублирование, российские аналоги (Iskra вместо Arduino) |
| **JLCPCB недоступен** | Нет прямой оплаты и доставки | Посредники (BuyChina, PochtaDirect) или РАСМО/Резонит |
| **Моторы Damiao только в Китае** | DM-J10010L, DM4340p | Посредники, групповые закупки |
| **CNC-обработка дорогая в РФ** | В 3-5 раз дороже Китая | Xometry, локальные цеха, посредники для PCBWay |
| **Подшипники специальные** | Карданные, шарнирные — редкие | Podshipnik.ru, SKF, заказ из Китая |
| **Ограничения ТК по весу** | Моторы тяжёлые | ПЭК, Деловые Линии для грузов >25 кг |
| **Аккумуляторы — опасный груз** | LiPo ограничения | Только наземная доставка, уточнять у ТК |
| **Сроки из регионов** | СПб, Пермь — 3-7 дней | Заказывать заранее, планировать буфер |
| **Курс валют** | Цены на импорт растут | Закупать партиями, фиксировать курс |
