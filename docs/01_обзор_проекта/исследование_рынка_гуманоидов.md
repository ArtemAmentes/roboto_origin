# Исследование рынка гуманоидных роботов: что продаётся, что — только разработка

Документ подготовлен для переговоров о закупке роботов в стартап. Содержит: какие модели **реально можно купить** (включая инвойсы), какие — только в разработке, возможности программирования и варианты самостоятельной сборки.

---

## Краткий вывод

**Индустрия гуманоидов в 2025–2026:** много анонсов, мало серийных продуктов с фиксированными ценами. Большинство — B2B, только по заявкам и через отдел продаж. Публичных прайс-листов почти нет. Редкие исключения: Unitree G1, Dobot Atom, Fourier GR-1.

## 2. Что в разработке или недоступно (инвойс получить сложно или невозможно)

### 2.1. Xiaomi CyberOne

- Заявленная цена: ~$70 000–$104 000  
- **Статус:** коммерческой продажи нет. Только R&D и внутреннее использование.  
- Планы массового производства на своих заводах к 2030. Внешних продаж не анонсировали.

### 2.2. Tesla Optimus

- Ориентир для Gen 3: $20 000–$30 000 при объёмах >1 млн/год  
- **Статус:** продажи внешним клиентам планируются с 2026. Сейчас — внутренние поставки на заводах Tesla.  
- Нет публичного канала для заказа.

### 2.3. RoboParty roboto_origin / Atom (Китай)

- **Статус:** открытый исходный код (GitHub: Roboparty/roboto_origin).  
- Железо: неясно, продаётся ли комплект или только документация и чертежи. Создатель собирал прототип AlexBot за ~$2 100.  
- Отдельный коммерческий канал с инвойсами не описан.

### 2.4. AGIBOT (Китай)

- Модели: A2 Ultra, A2 Lite, A2-W.  
- **Статус:** цены и условия поставки не обнародованы. Через демо и прямые переговоры.

## 3. Возможности программирования

### 3.1. Unitree

- **unitree_sdk2** (GitHub) — C++, есть Python-обёртка  
- Поддержка: Go2, B2, H1, G1  
- ROS2-пакеты, reinforcement learning (unitree_rl_gym), XR-телеоперация  
- ОС: Ubuntu 20.04 (aarch64, x86_64)

### 3.2. DeepRobotics

- J60/J80/J100: C-based SDK, CAN, визуальная отладка  
- Гуманоиды DR01/DR02: детали SDK на сайте не раскрыты, уточнять у продаж

### 3.3. Fourier

- Документация: support.fftai.cn (GRX, GR-1)  
- API и протоколы — в техдокументации и у партнёров

### 3.4. Dobot

- Закрытая экосистема, API уточнять у дилера (dobot.ru, Teswel)

## 4. Как собрать гуманоида самостоятельно

### 4.1. Модули суставов DeepRobotics

- **J60** — для образования и DIY. Высокое отношение крутящего момента к весу.  
- **J80 / J100** — для рук/ног крупных гуманоидов.  
- Можно проектировать свою конструкцию, докупать корпус, электронику, ПО.  
- Заказ: форма на сайте, partner@deeprobotics.cn.

### 4.2. Проект InMoov (open source)

- Конструкция для 3D-печати  
- Оценка: ~14 кг пластика, ~30 сервоприводов, 2× Arduino Mega, аккумуляторы, миникомпьютер, камера  
- BOM и инструкции: 3deshnik.ru, thingiverse.com  
- Оценка бюджета: от $2 000–5 000 (зависит от сервоприводов и доп. оборудования)

### 4.3. RoboParty / roboto_origin

- Открытая платформа, документация в репозитории  
- Оценка стоимости прототипа AlexBot: ~$2 100  
- Уточнять актуальность BOM и наличие комплектов у RoboParty

| Задача | Варианты |
|--------|----------|
| Быстрый доступ, фиксированная цена | **Unitree G1** — официальный магазин и дилеры |

**Для инвойса:**  
1. Unitree, Dobot (Teswel), NEURA, 1X — есть формальные каналы B2B.  
2. DeepRobotics, Fourier, AGIBOT — через форму «product purchase intention» или прямой контакт.  
3. Xiaomi, Tesla — внешних продаж нет.
