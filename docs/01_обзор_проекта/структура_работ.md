# Структура работ по робототехнике

Документ для внутреннего планирования команды.

---

## 1. Четыре направления работ

### Направление 1: Железо

- Импорт образцов и реверс-инжиниринг
- Каталогизация компонентов
- Сборка/ремонт/обслуживание
- Производство компонентов

### Направление 2: Софт

- SDK для управления робототехникой
- LLM-агент: ТЗ на человеческом языке → алгоритм движения
- Интеграция с MES/SCADA/ERP
- Автоматизация производств

### Направление 3: Интеграция

- CVCODE в роботах (персонализация, гражданская идентичность)
- ANIEMORE в роботах (распознавание эмоций в голосе)
- Пилоты в Москве (Депздрав, предприятия)

### Направление 4: Кадры и инфраструктура

- Формирование команды
- Сборочная мастерская, ЗИП
- Образование, колледж
- Кафедры робототехники в московских ВУЗах

---

## 2. Этап 1 (1 млрд) — экспериментальная лаборатория в Москве

| Направление | Что делаем | Результат |
|-------------|------------|-----------|
| **Железо** | Импорт образцов, реверс-инжиниринг, каталогизация. Экспериментальная сборка на импортных компонентах с заменой на российские аналоги. | Каталог компонентов, 1–3 прототипа, стратегия импортозамещения |
| **Софт** | Российский SDK для перепрошивки роботов. LLM-агент. API для MES/SCADA/ERP. Централизованное обновление и закрытый контур. | Открытый SDK, драйверы для 2–3 моделей, LLM-агент MVP |
| **Интеграция** | CVCODE и ANIEMORE в прошивке. Пилоты в Депздраве Москвы и на предприятиях. | Робот с CVCODE+ANIEMORE, 3–5 пилотных точек |
| **Кадры** | Формирование команды. Первая сборочная мастерская в Москве. | Команда 15–30 человек, мастерская с ЗИП |

**Критерий успеха:** рабочий прототип, открытый SDK, первые пилоты, команда и мастерская.

---

## 3. Этап 2 (9 млрд) — промышленное производство и масштабирование

| Направление | Что делаем | Результат |
|-------------|------------|-----------|
| **Железо** | Производственные линии компонентов и сборки роботов в Москве. Полный цикл. Запасы ЗИП. | Серийное производство, номенклатура с российскими аналогами |
| **Софт** | SDK для всего парка. Визуальный конструктор сценариев. Предиктивное обслуживание. Полная автоматизация производств. | Софт для автоматизации производств |
| **Интеграция** | Масштабирование пилотов на регионы. Филиалы завода. Региональные мастерские. | Сеть внедрений по России |
| **Кадры** | Кафедры в МФТИ, Бауманка, МАИ. Колледж. Стажировки. Поиск талантов через CVCODE. | Система подготовки кадров, Москва — центр притяжения |

**Критерий успеха:** серийное производство в Москве, кафедры в ВУЗах, филиалы в регионах.

## 4. Work packages (детализация)

### Железо

- WP-H1: Закупка и логистика импортных роботов
- WP-H2: Реверс-инжиниринг, протоколы, чертежи
- WP-H3: Каталог компонентов и roadmap замены на российские аналоги
- WP-H4: Сборочная мастерская: оборудование, процессы, ЗИП
- WP-H5 (Э2): Проектирование и запуск производственных линий
- WP-H6 (Э2): Производство компонентов и сборка роботов

### Софт

- WP-S1: Архитектура SDK, API, драйверы
- WP-S2: LLM-агент: prompt-инжиниринг, ТЗ → алгоритм
- WP-S3: Интеграция с MES/SCADA/ERP
- WP-S4: Визуальный конструктор сценариев (low-code)
- WP-S5: Мониторинг, предиктивное обслуживание, отчёты

### Интеграция (CVCODE, ANIEMORE)

- WP-I1: Встраивание CVCODE в прошивку
- WP-I2: Встраивание ANIEMORE
- WP-I3: Пилоты: выбор площадок, сценарии, метрики
- WP-I4 (Э2): Репликация пилотов в регионах

### Кадры и инфраструктура

- WP-K1: Рекрутинг и формирование команды
- WP-K2: Аренда/обустройство площадки в Москве
- WP-K3 (Э2): Переговоры с ВУЗами о кафедрах
- WP-K4 (Э2): Программы обучения и стажировок
- WP-K5 (Э2): CVCODE для поиска талантов
