# Закупка механических компонентов в России

Документ содержит конкретные ссылки и рекомендации по закупке механических компонентов из BOM робота ATOM 01 для российских условий.

---

## 1. Обзор проблемы

**Электроника** — ссылки есть (ChipDip, Амперка, Taobao).

**Механика** — ссылок нет:
- CNC-детали (алюминий 6061, пескоструй, анодирование)
- Подшипники (специальные: шарнирные, игольчатые упорные, крупногабаритные)
- Крепёж (ГОСТ/DIN, высокопрочный)
- Моторы Damiao (только Китай)

**Кастомные платы** — JLCPCB ушёл, нужны посредники.
