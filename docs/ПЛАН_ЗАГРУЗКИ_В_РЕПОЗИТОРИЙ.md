# Полный план загрузки проекта в приватный репозиторий

Детальный план интеграции проекта «Национальная платформа российской робототехники» в приватный GitHub-репозиторий [ArtemAmentes/roboto_origin](https://github.com/ArtemAmentes/roboto_origin).

---

## 1. Исходное состояние

### 1.1. Локальная папка РОБОТЫ

| Путь | Содержимое |
|------|-------------|
| `docs/` | 18 MD-файлов в 5 разделах |
| `Презентация/` | create_presentation.py, requirements.txt, фото (photo_*.jpeg), PNG (robots_components_transfer.png) |
| Git | Не инициализирован |

### 1.2. GitHub

| Репозиторий | Статус | Видимость |
|-------------|--------|-----------|
| ArtemAmentes/roboto_origin | Пустой (no commits) | **Приватный** — переключить перед загрузкой |
| Roboparty/roboto_origin | Оригинал робота, assets, modules | Публичный |

### 1.3. Цель

- Единый приватный репозиторий с полной документацией проекта
- Презентация и скрипт генерации .pptx
- План перевода roboto_origin на русский
- Связь с оригиналом roboto_origin (ссылки, без копирования кода робота в первый коммит)

## 6. Команды для выполнения

```bash
# 1. Перейти в рабочую папку
cd /Users/amentes/Desktop/roboto_origin_work   # или куда клонирован репо

# 2. Скопировать содержимое (если ещё не скопировано)
cp -R /Users/amentes/Desktop/РОБОТЫ/docs .
mkdir -p presentation
cp /Users/amentes/Desktop/РОБОТЫ/Презентация/*.py presentation/
cp /Users/amentes/Desktop/РОБОТЫ/Презентация/requirements.txt presentation/
cp /Users/amentes/Desktop/РОБОТЫ/Презентация/*.jpeg presentation/ 2>/dev/null || true
cp /Users/amentes/Desktop/РОБОТЫ/Презентация/*.png presentation/ 2>/dev/null || true

# 3. Добавить в репо
git add .
git status   # проверить

# 4. Коммит
git commit -m "Национальная платформа робототехники: документация, презентация, русский план"

# 5. Push (если репо уже приватный)
git push -u origin main
```
