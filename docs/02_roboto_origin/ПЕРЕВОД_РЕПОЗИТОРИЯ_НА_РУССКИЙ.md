# Перевод репозитория roboto_origin на русский язык

Документ описывает цель и объём работ по полной русификации репозитория [Roboparty/roboto_origin](https://github.com/Roboparty/roboto_origin).

---

## Цель

Создать собственный форк репозитория roboto_origin с **полностью русской локализацией**:

- Вся документация переведена на русский язык
- Все файлы (README, CONTRIBUTING, docstrings, комментарии в коде) — на русском языке
- Код остаётся функционально идентичным, меняется только текстовая составляющая

---

## Объём работ

### 1. Документация

| Файл / раздел | Действие |
|---------------|----------|
| README.md | Перевод на русский, сохранение структуры (Quick Start, модули, BOM) |
| README_cn.md | Заменить на README_RU.md как основной русский README |
| CONTRIBUTING.md | Полный перевод |
| CODE_OF_CONDUCT.md | Полный перевод |
| CODE_OF_CONDUCT_CN.md | Удалить или заменить русской версией |
| Humanoid Robot Know-How Documentation | Полный перевод всей R&D-документации |
| README в каждом sub-репозитории | atom01_hardware, atom01_deploy, atom01_train, atom01_description — все на русском |

### 2. Код

| Элемент | Действие |
|---------|----------|
| Docstrings (Python, C++) | Перевод всех описаний функций, классов, модулей на русский |
| Комментарии в коде | Перевод однострочных и многострочных комментариев на русский |
| Сообщения и строки пользовательского интерфейса | Перевод на русский (логи, ошибки, подсказки, если отображаются пользователю) |
| Имена переменных и функций | Оставить на английском (следование общепринятой практике программирования) |
| Строковые литералы в коде | Перевод на русский, если строка отображается пользователю |

### 3. Метаданные и конфиги

| Элемент | Действие |
|---------|----------|
| LICENSE | GPL v3 сохраняется без изменений (официальный текст лицензии) |
| .gitignore, .gitattributes | Без изменений (технические) |
| Конфигурационные файлы (YAML, JSON, etc.) | Комментарии и описательные поля — на русском, ключи — без изменений |

---

## Принципы

1. **Сохранение функциональности** — код работает идентично оригиналу.
2. **Сохранение лицензии** — GPL v3, указание на оригинальный проект RoboParty.
3. **Русский язык как основной** — для российских разработчиков и участников проекта.
4. **Имена в коде** — переменные, функции, классы остаются на английском для совместимости и читаемости в мировом контексте.

## Этапы реализации

1. **Форк и клонирование** — создание собственного репозитория на базе roboto_origin.
2. **Документация** — перевод README, CONTRIBUTING, Know-How в основном репо.
3. **Sub-репозитории** — аналогичный перевод во всех модулях.
4. **Код: docstrings** — перевод описаний в Python (deploy, train) и C++.
5. **Код: комментарии** — перевод комментариев в ключевых файлах.
6. **Проверка** — тесты, сборка, проверка что всё работает после изменений.

## Результат

Форк с полностью русской документацией и русифицированным кодом (документация и комментарии), готовый для использования российской командой разработчиков и сообществом.
