# Сборка робота Atom01 (roboto_origin)

Краткое руководство по сборке гуманоидного робота Atom01 на базе проекта [Roboparty/roboto_origin](https://github.com/Roboparty/roboto_origin).

---

## Quick Start

```bash
# Клонировать репозиторий
git clone https://github.com/Roboparty/roboto_origin.git
cd roboto_origin

# Обновить (при использовании submodules)
git pull

# Перейти в модули и следовать README
cd modules/Atom01_hardware   # механика, CAD, PCB
cd modules/atom01_deploy      # ROS2-драйверы
cd modules/atom01_train       # RL, Isaac Lab
cd modules/atom01_description # URDF, MuJoCo
```

## Модули

| Модуль | Описание |
|--------|----------|
| Atom01_hardware | Механика (atom01_mechnaic), печатные платы (Roboto_Power, Roboto_Usb2Can) |

| atom01_train | RL в Isaac Lab, Sim2Sim в MuJoCo |
| atom01_description | URDF, MJCF, меши, terrain_assets |
