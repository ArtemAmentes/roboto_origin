# Skill «управление роботом»

OpenClaw Skill — расширение, которое агент может вызывать как инструмент.

## Идея

Skill «управление роботом» предоставляет агенту API для:
- Отправки команд движения (поднять руку, повернуться, сделать шаг)
- Запроса состояния робота (позиция, сенсоры)
- Запуска сценариев (подойти к объекту, взять предмет)

## Структура Skill

1. **Описание для агента:** что умеет skill, какие параметры принимает
2. **Реализация:** вызовы к atom01_deploy / ROS2
3. **Безопасность:** ограничения, аварийная остановка, зоны безопасности

## Пример сценария

Пользователь в Telegram: «подойди к стулу и остановись в метре»

Агент:
1. Парсит команду
2. Вызывает skill с параметрами (цель: стул, расстояние: 1 м)
3. Skill генерирует траекторию / вызывает API навигации
4. Робот выполняет
