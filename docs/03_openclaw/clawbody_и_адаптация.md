# ClawBody и адаптация под atom01

## ClawBody

**ClawBody** (Tom Rikert) — программный мост, соединяющий OpenClaw с физическим железом.

Возможности:
- Подключение OpenClaw к реальным моторам
- Интеграция с MuJoCo — обучение агентов в симуляции перед выкладкой на железо
- Отладка в виртуальной среде без риска для оборудования

**Демонстрация:** подключение к гуманоиду Reachy Mini.

---

## Адаптация под roboto_origin / atom01

roboto_origin использует:
- ROS2
- atom01_deploy — драйверы, middleware, IMU, моторы
- Isaac Lab для симуляции

**Задачи адаптации:**
1. Изучить протокол управления atom01_deploy
2. Реализовать адаптер ClawBody → ROS2 / atom01_deploy
3. Настроить MuJoCo или Isaac Lab для обучения агентов
4. Обеспечить низкую задержку: команды агента → движение робота
