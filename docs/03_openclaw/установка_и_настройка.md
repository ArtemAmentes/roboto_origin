# Установка и настройка OpenClaw

## Требования

- Node.js 22+
- API-ключ от провайдера (Claude, OpenAI, DeepSeek и др.)
- 5–10 минут на первоначальную настройку

## Quick start

```bash
# Установка
npm install -g openclaw@latest

# Онбординг и установка сервиса
openclaw onboard --install-daemon

# Вход в каналы и запуск Gateway
openclaw channels login
openclaw gateway --port 18789
```

## Конфигурация

Файл: `~/.openclaw/openclaw.json`

Пример ограничения доступа:
```json5
{
  channels: {
    whatsapp: {
      allowFrom: ["+7XXXXXXXXXX"],
      groups: { "*": { requireMention: true } },
    },
  },
}
```

## Dashboard

- Локально: http://127.0.0.1:18789/
- Удалённый доступ: см. [документацию OpenClaw](https://docs.openclaw.ai/)
