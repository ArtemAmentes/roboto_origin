# Презентация для Правительства Москвы

Материалы для заявки на субсидию: национальная платформа робототехники.

---

## О презентации

**Цель:** получение субсидии на создание экспериментальной лаборатории (Этап 1) и промышленного производства (Этап 2) робототехники в Москве.

**Аудитория:** муниципальные чиновники, департаменты промышленности и здравоохранения Москвы.

**Ключевые акценты:**
- Москва — будущий центр российской робототехники (Силиконовая долина роботов)
- Все работы ведутся в Москве
- Доказанный опыт команды (Civicode, ANIEMORE, госконтракты)
- Софт для полной автоматизации производств (для Деппром)
