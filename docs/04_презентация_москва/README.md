# Презентация для Правительства Москвы

**Материалы для заявки на субсидию** — национальная платформа робототехники.

---

## О презентации

| Параметр | Значение |
|----------|----------|
| Цель | Субсидия ~10 млрд ₽ |
| Аудитория | Департаменты промышленности и здравоохранения Москвы |
| Этапы | Этап 1 (1 млрд) + Этап 2 (9 млрд) |

**Ключевые акценты:**
- Москва — будущий центр российской робототехники
- Все работы ведутся в Москве
- Доказанный опыт команды (Civicode, ANIEMORE, госконтракты)
- Софт для полной автоматизации производств
