## Подтверждённые продукты

### Civicode (Сивикод)

- Всероссийское открытое ПО для оцифровки личности человека
- Внедрение в Правительстве Москвы (Департамент здравоохранения), первые госконтракты
- Федеральные проекты: «Школьная проектная олимпиада», «Стройотряды России» и др.

### КОД-ИИ / ANIEMORE

- Опенсорсная ИИ-библиотека для распознавания эмоций в речи человека
- ВУЗы и РАН — решения в Санкт-Петербурге, Новосибирске, Севастополе
- МТС внедрила технологию в свои сервисы
- Стандарт для распознавания эмоций на русском языке
