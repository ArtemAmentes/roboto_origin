# Изменение скорости передачи IMU Hipnuc (CAN J1939)

Пример: изменение скорости устройства с 500k на 1M.

Убедитесь, что CAN-интерфейс настроен на текущую скорость устройства (например, 500k), затем отправьте команды:

## Шаг 1: Команда «изменить скорость на 1M»

- **ID:** `0CEF0800` (отправка на 08, от 00)
- **Данные:** `9A 00 06 00 00 00 00 00` (адрес 009A, команда записи 06, данные 0 = 1000K)

```bash
cansend can4 0CEF0800#9A00060000000000
```

## Шаг 2: Команда «сохранить конфигурацию»

- **Данные:** `00 00 06 00 00 00 00 00` (адрес 0000, команда записи 06, данные 0 = сохранить)

```bash
cansend can4 0CEF0800#0000060000000000
```

## Шаг 3: Команда «перезагрузка»

- **Данные:** `00 00 06 00 FF 00 00 00` (адрес 0000, команда записи 06, данные FF = сброс)

```bash
cansend can4 0CEF0800#00000600FF000000
```
